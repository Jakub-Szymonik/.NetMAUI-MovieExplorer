<h2>Purpose</h2>
I made an app that lets you browse movies by title or genra and save them to your watch history or favourites!

<h2>Features</h2>
<ul>
  <li>Search movies by title</li>
  <li>Filter movies by genre</li>
  <li>Display IMDB score</li>
  <li>Mark movies as watched</li>
  <li>Add movies to favourites</li>
  <li>Save user name</li>
  <li>View history with timestamps</li>
  <li>Data persistence (favourites and watched list saved locally)</li>
  <li>Dark/Light mode support</li>
</ul>

<h2>Technologies Used</h2>
<ol>
  <li>.Net Maui - Cross platform Framework</li>
  <li>C# - Primary programming language</li>
  <li>XAML - UI markup language</li>
  <li>JSON - Data storage and API response format</li>
</ol>
